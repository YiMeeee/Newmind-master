## 📦 一键下载脚本

### Python 下载 UCI 数据集

```python
import pandas as pd

# 下载 Iris 数据集
url = "https://archive.ics.uci.edu/static/public/4/data.csv"
df = pd.read_csv(url)
df.to_csv("iris.csv", index=False)

# 下载 Wine 数据集
url = "https://archive.ics.uci.edu/static/public/10/data.csv"
df = pd.read_csv(url)
df.to_csv("wine.csv", index=False)

# 下载 Breast Cancer 数据集
url = "https://archive.ics.uci.edu/static/public/17/data.csv"
df = pd.read_csv(url)
df.to_csv("breast_cancer.csv", index=False)
```

### Bash 下载 FiveThirtyEight 数据

```bash
# 克隆整个仓库
git clone https://github.com/fivethirtyeight/data.git

# 或下载单个文件
wget https://raw.githubusercontent.com/fivethirtyeight/data/master/college-majors/recent-grads.csv
```

### Python 下载 TensorFlow 数据集

```python
import tensorflow as tf

# MNIST
(x_train, y_train), (x_test, y_test) = tf.keras.datasets.mnist.load_data()

# Fashion MNIST
(x_train, y_train), (x_test, y_test) = tf.keras.datasets.fashion_mnist.load_data()

# IMDB Reviews
(x_train, y_train), (x_test, y_test) = tf.keras.datasets.imdb_reviews.load_data()

# Boston Housing
(x_train, y_train), (x_test, y_test) = tf.keras.datasets.boston_housing.load_data()
```

## 📌 使用提示

1. **UCI 数据集**: 所有链接格式为 `https://archive.ics.uci.edu/static/public/{ID}/data.csv`
2. **FiveThirtyEight**: 使用 GitHub Raw 链接直接下载 CSV
3. **TensorFlow 数据集**: 使用 `tf.keras.datasets` API 或直接下载
4. **大文件**: 建议使用 `wget` 或 `curl` 命令行下载
5. **MOOC 课程**: 中国大学 MOOC 和国家智慧教育平台可免费学习
