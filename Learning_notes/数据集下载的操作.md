## 大体操作如下
### 第一部分：环境与目录准备（只需执行一次）

这部分操作确保数据存放路径正确，且你的账号拥有完整的读写权限。

Bash

```
# 1. 创建位于 7.3T 大容量硬盘上的数据存放目录
sudo mkdir -p /data01/RNewmind_Datasets

# 2. 将目录所有权交给当前用户，防止后续下载出现权限错误
sudo chown -R $USER:$USER /data01/RNewmind_Datasets

# 3. 安装加速下载所需的工具包
pip install -U "huggingface_hub[cli]" hf_transfer
```

- **功能：** 解决物理空间不足和权限被拒绝（Permission Denied）的问题。
    

---

### 第二部分：开启“防掉线”后台会话

下载 3TB 数据可能需要数十小时，直接在 SSH 窗口运行容易因网络波动中断。使用 `tmux` 可以让下载在后台持续运行。

Bash

```
# 1. 开启一个名为 "data" 的后台会话
tmux new -s data

# 2. 进入会话后，先开启 Rust 高速传输引擎环境变量
export HF_HUB_ENABLE_HF_TRANSFER=1
```

- **功能：** 保证你关闭电脑后，服务器依然在通宵下载；同时激活最高带宽下载模式。
    

---

### 第三部分：核心数据集下载指令

在 `tmux` 会话内，依次（或根据需求）执行以下指令。

#### 1. 预训练数据（第一阶段：核心基础知识）

Bash

```
# 下载 Cosmopedia (海量教科书与合成数据)
huggingface-cli download HuggingFaceTB/cosmopedia \
    --repo-type dataset \
    --local-dir /data01/RNewmind_Datasets/cosmopedia \
    --local-dir-use-symlinks False

# 下载 Arxiv (全量学术论文数据)
huggingface-cli download arxiv/arxiv \
    --repo-type dataset \
    --local-dir /data01/RNewmind_Datasets/arxiv \
    --local-dir-use-symlinks False
```

#### 2. SFT 监督微调数据（第二阶段：指令遵循能力）

Bash

```
# 下载 UltraData-Math (高质量数学推导数据)
huggingface-cli download openbmb/UltraData-Math \
    --repo-type dataset \
    --local-dir /data01/RNewmind_Datasets/ultradata_math \
    --local-dir-use-symlinks False

# 下载 OASST1 (多轮对话数据)
huggingface-cli download OpenAssistant/oasst1 \
    --repo-type dataset \
    --local-dir /data01/RNewmind_Datasets/oasst1 \
    --local-dir-use-symlinks False
```

#### 3. DPO 偏好对齐数据（第三阶段：价值观与逻辑优化）

Bash

```
# 下载 Orca DPO Pairs (人类偏好对齐数据)
huggingface-cli download Intel/orca_dpo_pairs \
    --repo-type dataset \
    --local-dir /data01/RNewmind_Datasets/orca_dpo \
    --local-dir-use-symlinks False
```

### 第四部分：常用维护指令（随时查阅）

| **功能**       | **指令**                               |
| ------------ | ------------------------------------ |
| **暂时离开后台会话** | 按下 `Ctrl + B`，然后快速按 `D`              |
| **重新回到下载界面** | `tmux attach -t data`                |
| **查看实时下载进度** | `du -sh /data01/RNewmind_Datasets/*` |
| **监控硬盘剩余空间** | `df -h /data01`                      |
