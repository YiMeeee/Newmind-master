在重构与修改完代码后，把整个项目的代码整理的可观一些。将minimind改名为RNewMind，注意一些必要的函数也需要进行替换
主要升级模块
一、NewMind 升级版核心技术模块
核心模块	技术组成	作用
Embedding 模块	Token Embedding	将文本 token 转换为向量
Position Encoding 模块	RoPE / NTK-RoPE	表示序列位置
Attention 模块	MHA / GQA / FlashAttention	建模 token 之间关系
FeedForward 模块	SwiGLU FFN	非线性特征变换
Normalization 模块	RMSNorm	稳定训练
MoE 模块	Mixture of Experts	扩展模型容量
Training Engine	优化器 + scheduler	模型训练
Inference Engine	KV Cache + Sampling	文本生成

这 8 个模块共同构成了 完整的大语言模型系统核心。

二、Transformer 核心结构模块

在模型内部，最核心的结构是 Transformer Block。

结构如下：

Input
↓
Embedding
↓
Transformer Block × N
↓
LM Head
↓
Output

每个 Transformer Block 由三个关键技术模块组成：

模块	技术
Attention	自注意力机制
FeedForward	SwiGLU
Normalization	RMSNorm

这是目前主流 LLM 的基础结构。

例如：

GPT-4

Claude

都基于类似结构。

三、Attention 模块（最核心模块）

Attention 是 Transformer 的核心计算模块。

主要功能：

建立 token 之间的依赖关系

处理长文本上下文

NewMind 升级支持三种注意力结构：

技术	作用
Multi-Head Attention	标准注意力
Grouped Query Attention	减少显存
Multi Query Attention	推理优化

同时支持：

FlashAttention 加速算法。

FlashAttention 在 GPU（例如 NVIDIA A100）上可以显著提高训练速度并降低显存使用。

四、RoPE 位置编码模块

Transformer 本身无法感知 token 的顺序，因此需要位置编码。

NewMind 使用：

RoPE（Rotary Position Embedding）

优势：

适合长上下文

可扩展

升级支持：

技术	作用
NTK RoPE	扩展上下文
YaRN RoPE	稳定长序列
五、FeedForward 模块

FeedForward 是 Transformer 中第二大计算模块。

NewMind 升级为：

SwiGLU 结构。

公式：

FFN(x) = W2( SiLU(W1x) * W3x )

相比传统 FFN：

传统FFN	SwiGLU
ReLU	SiLU
2层	3权重矩阵
表达能力较弱	表达能力更强

很多现代模型使用该结构，例如：

Google PaLM

DeepSeek 系列模型

六、Normalization 模块

Normalization 用于稳定训练。

NewMind 默认使用：

RMSNorm。

优点：

优势	描述
计算更简单	不需要均值
训练稳定	减少梯度爆炸
速度更快	更适合大模型

相比：

LayerNorm 更轻量。

七、MoE 模块（专家网络）

MoE 是扩展模型容量的重要技术。

结构：

Input
↓
Router
↓
Select Experts
↓
Expert Networks
↓
Output

核心机制：

技术	作用
Router	选择专家
Top-k routing	选择k个专家
Load balance	平衡计算

MoE 可以让模型拥有：

巨大参数量

低计算成本

例如：

Google Switch Transformer

DeepSeek MoE 模型

八、训练系统模块

训练系统负责模型训练流程。

核心组件：

Trainer Engine。

功能：

功能	描述
Forward	前向计算
Loss	损失函数
Backward	反向传播
Optimizer	参数更新
Checkpoint	模型保存

支持优化器：

优化器
AdamW
Lion
Adafactor

学习率策略：

Warmup

Cosine decay

九、推理系统模块

推理系统用于生成文本。

核心技术：

技术	作用
KV Cache	减少重复计算
Sampling	控制生成
Top-k	限制候选词
Top-p	概率采样

推理流程：

Prompt
↓
Tokenizer
↓
Model
↓
Sampler
↓
Text
